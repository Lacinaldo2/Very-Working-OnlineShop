# Projekt Alegro

To jest prosty sklep internetowy z obsługą urzytkowników, koszyka oraz historii zamówień.

## Użyliśmy:
- React
- Vite
- JavaScript
- Context API
- CSS
- Mockowane API (dane w plikach JSON)

## Opis funkcjonalności
Aplikacja umożliwia:
- wyświetlanie listy produktów
- przechodzenie do szczegółów produktu
- dodawanie produktów do koszyka
- usuwanie produktów z koszyka oraz przeliczanie ceny
- składanie zamówień
- podgląd historii zamówień i szczegółów zamówienia
- logowanie i rejestrację użytkowników (dane mockowane)
- dodawanie opinii o produktach

## Uruchomienie projektu
npm install
npm run dev

## Uwagi
- Projekt korzysta z mockowanych danych.
- Logowanie odbywa się na podstawie danych zapisanych w plikach JSON.
